# NOMAAD Catering — Хоолны технологи & нөөц

Чимун ХХК-ийн **NOMAAD Catering** (B2B арга хэмжээний кейтеринг)-ийн хоолны орц,
технологи, нөөц, **зардлын** интерактив веб апп. Хүний тоо, идэх хандлага, эр/эм
харьцааг оруулахад 10 хоолны түүхий хэрэгцээ, нөөц, өртөг автоматаар бодогдоно.

> **Хувилбар:** Порц стандарт **v1.0** · © 2026 Чимун ХХК · NOMAAD family

## Онцлог

- **Интерактив тооцоолуур** — нийт хүн, ширээний хүн оруулахад бүх хэмжээ шинэчлэгдэнэ
- **10 үндсэн хоол** — буфет / порц / шэйр үйлчилгээний төрлөөр
- **Махны төрлөөр cook yield** — хавтгай тоо биш, мах тус бүрийн жинхэнэ гарцаар (0.60–0.80)
- **6 түвшний идэх хандлага** — албан хурлаас (×1.00) BBQ үдэш (×1.65) хүртэл
- **Эр/эм харьцаа** — оруулахад хэрэгцээ бага зэрэг тохирно (эр ~+10%, эм ~−10%)
- **Зардлын тооцоо** — мах ₮/кг + дагалдах ₮/хүн оруулахад хоол бүрийн өртөг, нэг хүний зардал, нийт зардал
- **Зорилт финиш** — нэг хүнд тавагт хэдэн г буухыг харуулна
- **Фильтр** — Буфет дарвал зөвхөн буфет, Порц дарвал зөвхөн порц
- **Хөлдөөх арга** — ★ түүхий хөлдөөх, ☆ болгож хөлдөөх, шинэ
- **Нөөц хяналт** — одоогийн нөөцөө оруулахад дутагдал/илүүдэл шууд харагдана (хөтөч дотор хадгалагдана)
- **Эрүүл ахуй (HACCP)** — CCP критик цэгүүд улаанаар тэмдэглэгдэнэ
- Утас, таблет, компьютерт тохирсон responsive дизайн

## Тооцооны логик (v1.0)

Түүхий хэрэгцээ дараах томьёогоор бодогдоно:

```
Түүхий (кг) = Хүн × Идэх коэф. × Эр/эм коэф. × Норм(г) ÷ Cook yield × (1 + буфер)
```

### Cook yield — махны төрлөөр

| Мах төрөл | Yield | Тэмдэглэл |
|---|---|---|
| Тахианы цээж | 80% | Зөөлөн шарах |
| Тахианы гуя | 74% | Арьстай хэсэг |
| Хонь (зөөлөн жигнэх) | 70% | Соустай удаан жигнэх |
| Хонь ястай (хавирга) | 60% | Ясны жин + чийг алдалт |
| Гахай цорой | 78% | Цэвэр булчин |
| Гахай (амталсан) | 74% | — |
| Гахай ястай | 62% | — |
| Холимог | 76% | Жигнэсэн дундаж |

### Идэх хандлагын коэффициент

| Арга хэмжээ | Коэф. |
|---|---|
| Албан хурал (дотор) | ×1.00 |
| Team-building (дотор) | ×1.15 |
| Гадаа + энгийн | ×1.30 |
| Гадаа + пиво | ×1.45 |
| Том найр / Хурим | ×1.50 |
| BBQ үдэш | ×1.65 |

### Бусад тогтмол

| Үзүүлэлт | Утга |
|---|---|
| Буфет буфер | +20% |
| Порц буфер | +8% |
| Эр/эм коэф. | эр ×1.10, эм ×0.90 (харьцаагаар) |
| Уут | 30×40 см / 2 кг |

> **Стандартын статус:** HACCP температурууд (≤4°C, ≥75°C, blast chill) — олон улсын
> баталгаатай. Cook yield нь USDA + industry, идэх коэф. ба эр/эм коэф. нь industry
> rule of thumb (батлагдаагүй v1.0 эхлэх тоо). 100+ арга хэмжээний бодит өгөгдөл
> цуглуулсны дараа v1.1 болгож тохируулна.

## Тохируулга

Бүх норм, хоол, yield, коэффициент `index.html` доторх `<script>` хэсэгт байна:

- `DISHES` — 10 үндсэн хоол (нэр, уураг, yieldKey, норм, орц, технологийн алхам)
- `SIDES` — дагалдах хоолны норм
- `COOK_YIELD` — махны төрлөөр гарц
- Идэх товчнууд — `data-a` утгаар (1.00–1.65)
- Зардлын үнэ, цэсийн сонголт, нөөц — хөтчийн `localStorage`-д хадгалагдана
